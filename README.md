# Mini-Processor-Verilog

A minimal educational single-cycle processor skeleton showing datapath and simple instruction execution.

## Files
- `mini_processor.v` - simplified RTL
- `mini_processor_tb.v` - testbench skeleton

## Notes
This is a learning project: extend the instruction set, add control unit, and test with instruction memory initialization.
